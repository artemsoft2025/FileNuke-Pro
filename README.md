# FILE NUKE PRO 1.0

Данная программа предназначена для **намеренного повреждения файлов**, а не для их мгновенного удаления или классического «затирания».

После обработки файл:
- **перестаёт открываться** стандартными программами (видео-плеерами, просмотрщиками изображений, офисными приложениями);
- **теряет корректную структуру**, необходимую для восстановления;
- может быть **безопасно удалён позже** — вручную или с помощью других средств.

Программа ориентирована на **обычных пользователей**, которым важно визуально убедиться, что данные больше не пригодны к использованию.

## В чём проблема обычных программ удаления

Многие программы, заявляющие «безвозвратное удаление», на практике:
- не учитывают особенности **USB-флэшек и SSD** (wear leveling, кэширование, перераспределение блоков);
- затирают** логические адреса**, но не всегда реальные физические ячейки памяти;
- оставляют файл **структурно целым**, из-за чего его можно восстановить специализированными утилитами.

В результате пользователь уверен, что файл уничтожен, но на самом деле:
- видео можно восстановить;
- фото снова открывается;
- документ читается частично или полностью.

---

## Подход, который использует эта программа

Вместо попыток «гарантированно стереть всё сразу», программа применяет **разрушение файловой структуры**.

Проще говоря:
- файл **ломается изнутри**;
- критические участки данных становятся неконсистентными;
- стандартные и массовые утилиты восстановления **не способны вернуть файл в рабочее состояние**.

При этом:
- файл **остаётся на носителе**;
- пользователь может сам проверить результат;
- удаление можно выполнить позже, любым удобным способом.

---

## Что происходит при обработке

1. Анализ файла - определение типа и критических областей
2. Порча данных - целенаправленное повреждение критических байтов
3. Добавление шума - случайные данные для маскировки
4. Переименование - если включен параметр -R
5. Верификация - проверка качества порчи (если не отключено)

---

## Почему это безопаснее и надёжнее

**1. Видимый результат**

Пользователь сразу видит:
- видео не воспроизводится;
- фото не открывается;
- документ вызывает ошибку.

Нет «веры на слово» — всё проверяется вручную.

**2. Минимум иллюзий**

Программа не обещает «100% физическое уничтожение», а делает то, что действительно работает:

файл становится бесполезным.

**3. Защита от массового восстановления**

Обычные программы восстановления:
- ориентированы на **целые сигнатуры файлов**;
- не умеют корректно восстанавливать намеренно повреждённую структуру.

## Примеры использования

**Пример 1. Видео на флэшке**
Вы скопировали личное видео на USB-накопитель и хотите избавиться от него перед передачей флэшки другому человеку.
1.	Обрабатываете файл программой.
2.	Проверяете — видео не воспроизводится.
3.	Удаляете файл обычным способом.
Даже при попытке восстановления файл не станет рабочим видео.
________________________________________
**Пример 2. Фотографии**
Есть фотографии, которые не должны быть восстановлены.
1.	Запускаете программу.
2.	Обрабатываете выбранные изображения.

4.	Удаляете.
________________________________________
**Пример 3. Документы**
Документы с конфиденциальной информацией:
- отчёты

- договоры

- текст невозможно восстановить стандартными средствами.

## Рекомендуемый порядок действий

1.	**Обработать файл программой**
2.	**Проверить**, что файл не открывается
3.	**Удалить файл**
4.	(Опционально) выполнить дополнительное затирание свободного места другой программой

## Важно понимать

- Программа **не является криптографическим уничтожителем**.
- Она предназначена для **практической защиты от восстановления**, а не для борьбы с лабораторной криминалистикой.
- Для большинства бытовых и офисных сценариев этого **более чем достаточно**.

## Параметры программы

**Основные параметры:**
```cmd
-i, --intensity FLOAT   Интенсивность порчи (0.05-0.5, по умолчанию 0.2)
-r, --recursive         Рекурсивная обработка подпапок
-R, --rename            Переименовать файлы после порчи
-f, --force             Принудительно обрабатывать системные файлы
-t, --keep-time         Сохранить оригинальное время файла
-q, --quiet             Тихий режим (минимальный вывод)
--test                  Тестовый режим (не портит файлы)
```

**Дополнительные параметры:**
```
--no-verify             Отключить проверку результата
--no-noise              Не добавлять шумовые данные
-h, --help              Показать эту справку
```
