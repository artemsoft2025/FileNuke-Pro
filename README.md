# В чем уникальность FileNuke Pro

FileNuke Pro - это **гибридный** инструмент, который сочетает два подхода:

1. **Умная структурная порча** - анализирует тип файла и целенаправленно разрушает его критичные области
2. **Полное затирание** - реализует стандарты безопасного удаления данных

В отличие от других программ:

- Видимый результат - файл можно проверить ДО удаления
- Интеллектуальный анализ - программа "понимает", что портить в разных типах файлов
- Гибридный подход - можно использовать порчу ИЛИ затирание ИЛИ оба подхода
- Безопасность - встроенные проверки защищают от случайного повреждения системных файлов
- Параллельная обработка - быстро работает с большими наборами файлов

# Примеры использования

```cmd
Пример 1: Быстрая порча временных файлов
filenuke.exe "C:\temp\*.tmp" -i 0.4 -R --parallel
Интенсивная порча с переименованием, параллельная обработка

Пример 2: Уничтожение конфиденциальных документов
filenuke.exe "секретные_документы/" -r --two-pass --passes 5
Рекурсивная обработка, двухэтапный режим с 5 проходами затирания

Пример 3: Безопасная очистка USB-накопителя
bash
filenuke.exe "D:\private_data\" --wipe-only --standard dod --passes 7
*Военный стандарт DoD 5220.22-M, 7 проходов затирания*

Пример 4: Выборочное уничтожение
bash
filenuke.exe "*.jpg" --intensity 0.3 --keep-time
Порча только JPG файлов с сохранением дат создания
```

# FAQ

A: Да, но учитывайте, что физическое уничтожение на SSD сложнее из-за wear leveling.

В чем отличие от подобных программ вкратце:

FILE NUKE позволяет испортить файл, в результате программы для восстановления данных (R.saver,  R-Studio, Recuva) не смогут восстановить эти файлы. Возможно дорогая криминалистика и поможет восстановить файлы удаленные данным способом, но главный смысл тут в том, что рядовые пользователи не смогу восстановить ваша данные без огромного вложения финансовых средств (взамен получаем экономию времени и ресурсов диска).

Конечно, txt файлы после порчи можно будет изучить почти полностью, но архивы, видео, фото, документы будут безнадежно испорчены. Но не забывайте, что теперь программа также может полностью затереть файлы - тут уже без вариантов для восстановления.

И filenuke не удаляет файл после затирания - пользователь может убедиться в эффективности и удалить файл обычным способом. 

Для надёжного удаления данных с флешки недостаточно просто стереть файлы и заполнить свободное место. Этот процесс долог и изнашивает накопитель. Альтернатива — затирание самого файла — также не даёт гарантии из-за особенностей флеш-памяти. Программа для стирания может показать успешное выполнение, но её обманут кэширование операционной системы и внутренний механизм выравнивания износа контроллера флешки, который перенаправляет запись в разные физические ячейки. Таким образом, следы файла могут остаться в памяти.

Поэтому эта программа поможет:
1. Когда нет времени/жалко диск. Файлы будут испорчены быстро и надежно.
2. Параноидальная защита - испортить файл, а потом уничтожить его другой популярной программой или параметр --two-pass в filenuke.

## В чём проблема обычных программ удаления

Многие программы, заявляющие «безвозвратное удаление», на практике:
- не учитывают особенности **USB-флэшек и SSD** (wear leveling, кэширование, перераспределение блоков);
- затирают** логические адреса**, но не всегда реальные физические ячейки памяти;
- оставляют файл **структурно целым**, из-за чего его можно восстановить специализированными утилитами.

В результате пользователь уверен, что файл уничтожен, но на самом деле:
- видео можно восстановить;
- фото снова открывается;
- документ читается частично или полностью.

---

## Подход, который использует эта программа

Вместо попыток «гарантированно стереть всё сразу», программа применяет **разрушение файловой структуры**.

Проще говоря:
- файл **ломается изнутри**;
- критические участки данных становятся неконсистентными;
- стандартные и массовые утилиты восстановления **не способны вернуть файл в рабочее состояние**.

При этом:
- файл **остаётся на носителе**;
- пользователь может сам проверить результат;
- удаление можно выполнить позже, любым удобным способом.

## Что происходит при обработке

1. Анализ файла - определение типа и критических областей
2. Порча данных - целенаправленное повреждение критических байтов
3. Добавление шума - случайные данные для маскировки
4. Переименование - если включен параметр -R
5. Верификация - проверка качества порчи (если не отключено)

## Почему это безопаснее и надёжнее

**1. Видимый результат**

Пользователь сразу видит:
- видео не воспроизводится;
- фото не открывается;
- документ вызывает ошибку.

Нет «веры на слово» — всё проверяется вручную. Просмотр файла hex-редактором позволяет убедиться, что код файла безнадежно испорчен. 

**2. Минимум иллюзий**

Программа не обещает «100% физическое уничтожение», а делает то, что действительно работает:

файл становится бесполезным.

**3. Защита от массового восстановления**

Обычные программы восстановления:
- ориентированы на **целые сигнатуры файлов**;
- не умеют корректно восстанавливать намеренно повреждённую структуру.

## Параметры программы

